# Lens: foundation-rewrite

When the formalization cannot host the needed object, change an a priori (coordinates, type universe, number system, what counts as a unit of the domain) instead of declaring impossibility inside the old frame.

## Questions that fire it

- Which foundational choice is doing the blocking — and what alternative choice makes the blocked object well-typed?
- What theorems of the old frame survive the rewrite, and which were only artifacts of the old packaging?
- Is the rewrite motivated by an exhibit path, or by escape from hard work without a new test?
