# Lens: counterexample-first

Before expanding a hopeful construction, hunt the smallest object that would kill it.

## Questions that fire it

- What is the cheapest counterexample class (numeric, finite, low-dimension, degenerate parameter)?
- Which assumption, if dropped, makes the claim false — and is that assumption load-bearing or cosmetic?
- If no counterexample is imaginable, is the claim vacuous rather than strong?
