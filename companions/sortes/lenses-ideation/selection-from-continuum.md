# Lens: selection-from-continuum

Discrete selected objects (special values, outliers, critical points, discrete solutions) vs continuous families the method only parametrizes — name the selector.

## Questions that fire it

- What continuous family is in play, and what discrete set is claimed to sit inside it?
- What rule *selects* the discrete set (combinatorial, extremal, positivity, external table, arithmetic) — is that rule inside the method or smuggled in packaging?
- Is an “eigenvalue / phase / spectral / ranking coincidence” actually a selection problem wearing continuum clothing?
