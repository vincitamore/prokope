# Lens: random-model-first

Replace the deterministic object with a random model that shares its coarse statistics, prove the target for the model, then measure exactly where the real object deviates. Structure is what survives after randomness is accounted for — the deviation IS the content.

## Questions that fire it

- What random stand-in matches your object's density and moments (random signs, random walks, shuffled labels, random points) — and is the target statement true for the model, almost surely?
- Where does the real object measurably depart from the model — and is that departure a nuisance, or the very structure the target is about?
- If the model version is provable, which property of the real object would let the proof survive de-randomization?

## Overfire (vice)

- Using the model's success as evidence about the real object without locating the deviation (the model is a map, not the territory).
- Randomness used only as a debunking control — this lens is generative: the model is a provable cousin, not just a null.
