# Lens: coordinate-invention

Invent a new coordinate or chart variable on the *same* objects when the current coordinates hide the joint — not only rewrite the whole foundation.

## Questions that fire it

- What quantity (rate, rank, latency tier, log-scale, residue, density) is doing work informally but is not a first-class coordinate?
- After defining it, which claims become definitions, which become theorems, and which die?
- Is this a true new axis, or a monotone relabel of an existing one (then prefer transport-vs-essential)?
