# Lens: construct-the-missing

When the work says a required structure is absent, treat absence as a construction order — not a stop.

## Questions that fire it

- What exact object is missing (definition, measure, map, generator), named so it could be built?
- What is the smallest interface that object would need to expose for the next claim to become checkable?
- If you cannot build it yet, what prototype or stand-in would still force a new experiment rather than a closure?
