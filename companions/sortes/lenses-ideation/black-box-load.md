# Lens: black-box-load

Separate what this work proves from load carried by an external black box (named theorem, table, oracle, unproved hypothesis, or foreign system).

## Questions that fire it

- Which steps of the argument are new here, and which silently invoke a named external theorem, hypothesis, or data table?
- If the black box were removed or replaced by a weaker lemma, what claim still stands on its own exhibits?
- Are we packaging “under hypothesis H / given table T / given system S” as if the box were contributed by this construction?
