# Lens: range-disjunction

Do not take the mode of the prior as the measure; range the disjunction of resolution paths and credit only exhibited ones.

## Questions that fire it

- What full set of paths could resolve this — including paths the default model never samples?
- Which path is popular because it is familiar rather than because it has the best exhibit so far?
- What single check would force the prior’s favorite path to compete with an under-sampled alternative?
